# Grub2Go

This is a simple Food Ordering Application made using Streamlit - Python.

### Prerequisites:
- Python 3.12 or above
- Streamlit should be installed:
    `pip install streamlit`

### Deliverables:
- app.py:
    Handles the order logic, validates user fields
- Menu.json:
    Allows the application owner to edit the menu card without altering the code. Must be in JSON format.
